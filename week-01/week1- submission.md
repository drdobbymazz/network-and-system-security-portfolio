## Action Plan

### 1. CompTIA Security+ Certification (Priority: High)
**Timeline**: 12 weeks | **Cost**: ¬£430

Study plan: Professor Messer videos + Darril Gibson book, 2 hours/day for 6 weeks, then practice exams until consistently scoring 85%+. Schedule exam for late March 2025.

**Why this matters**: Industry-standard validation of security knowledge. Required/preferred for both security roles. Demonstrates serious commitment beyond coursework.

### 2. Practical Security Tools (Priority: Critical)
**Timeline**: 12 weeks | **Cost**: ¬£30

Set up home lab (Kali Linux, Metasploitable). Learn Wireshark, Nmap, Burp Suite, Metasploit. Complete 15+ HackTheBox/TryHackMe challenges. Install SIEM (Splunk/ELK) and practice log analysis.

**Why this matters**: Biggest gap for security roles. Can't just know theory - need hands-on demonstration of tool proficiency.

### 3. Cloud Fundamentals - Azure (Priority: Medium)
**Timeline**: 10 weeks | **Cost**: ¬£0-80

Complete Azure Fundamentals learning path. Deploy TuneHub to Azure with proper security (HTTPS, authentication, Key Vault). Set up CI/CD with GitHub Actions.

**Why this matters**: Cloud is baseline expectation now. Addresses both FactSet infrastructure needs and modern security requirements.
