# Week 5 Code

Web security vulnerability scanning examples.

## Files

**wapiti_scanner.py** - Automated Wapiti scanner for authorized targets

Usage:
```bash
python wapiti_scanner.py
```

**scan_results.txt** - Example scan output showing 8 vulnerabilities found

## Requirements

```bash
pip install wapiti3
```

## Ethical Use Only

Only scan authorized training applications:
- Google Gruyere (your own instance)
- OWASP Juice Shop
- Never scan without explicit permission
